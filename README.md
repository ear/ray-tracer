# Ray Tracer in pure (99.999%) C++

![](https://i.imgur.com/WyFWDAP.png)

## Quick Start

```console
$ nix-shell         # Only if you are on NixOS
$ make              # To only build and lint the application
$ make trace        # To run the ray tracer (also builds the project)
$ feh output.png
```
